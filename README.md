# Customer Segmentation using K-Means Clustering

## SkillCraft Technology - Machine Learning Internship Task 2

### Objective
Develop a K-Means Clustering model to segment mall customers based on:

- Annual Income (k$)
- Spending Score (1–100)

### Dataset
Mall Customers Dataset (Kaggle)

### Libraries Used
- pandas
- numpy
- matplotlib
- scikit-learn

### Method
- Data preprocessing
- Elbow Method to determine the optimal number of clusters
- K-Means clustering
- Customer segmentation visualization

### Output
The repository contains:
- Python source code
- Elbow Method graph
- Customer Segmentation graph
- Dataset
